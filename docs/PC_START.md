# PC Start

## Clone

```powershell
cd C:\Users\morim
git clone https://github.com/morimilpabfelon-cell/Morimil-app.git
cd Morimil-app
```

## Apply Package Manually

If this package is used before all files are pushed to GitHub, copy the package contents into the cloned folder.

## Commit

```powershell
git status --short
git add .
git commit -m "feat: add Morimil app phase 1 software foundation"
git push origin main
```

## Build Later

Android Studio validation happens after the repo software foundation is complete.
